# Java-Quiz-App

<br>

## Features  
- GUI-based quiz interface  
- User inputs (Name and Roll Number)  
- 8 multiple-choice questions  
- Score calculation and result display  
- Scrollable quiz layout for better usability  

## Technologies Used  
- Java  
- Swing (JFrame, JPanel, JLabel, JButton, JRadioButton)  
- AWT  

## Installation and Usage  

### Prerequisites  
Ensure you have the following installed:  
- Java Development Kit (JDK)  
- Any IDE that supports Java (e.g., IntelliJ IDEA, Eclipse, or NetBeans)  

### Steps to Run  
  1. Clone the repository:  
   ```bash
   git clone https://github.com/Vedashree14/Java-Quiz-App.git
   ```
  2. Navigate to the project directory:
  ```bash
  cd Java-Quiz-App
  ```
  3. Compile the Java file:
  ```bash
  javac Quiz.java
  ```
  4. Run the application:
  ```bash
  java Quiz
  ```
